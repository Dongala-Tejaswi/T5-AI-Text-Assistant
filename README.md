T5 AI Text Intelligence Assistant

An AI-powered web application built using the T5 (Text-to-Text Transfer Transformer) model that performs multiple NLP tasks such as summarization, grammar correction, and question answering.

🚀 Features
✨ Text Summarization
✍️ Grammar Correction
❓ Question Answering
🌐 Interactive Web UI using Streamlit
🧠 Model Used
T5 (Text-to-Text Transfer Transformer)
Hugging Face Transformers
⚙️ Tech Stack
Python
Transformers (Hugging Face)
PyTorch
Streamlit

▶️ How to Run
1. Clone the Repository
git clone https://github.com/Dongala-Tejaswi/T5-AI-Text-Assistant
cd T5-AI-Text-Assistant
2. Install Dependencies
pip install -r requirements.txt
3. Run the App
python -m streamlit run app.py
